# Desktop build resources

electron-builder reads this directory (`directories.buildResources`, the
default `build/`) for application icons.

## Icons

- `icon.png` - 512x512 master, used for Linux and as the source
  electron-builder converts to macOS `.icns` at build time.
- `icon.ico` - multi-size Windows icon (16/24/32/48/64/128/256).

Both are the standard Angband icon (`reference/lib/icons/att-512.png`, the
upstream 4.2.6 art) with the mascot's original shield device (a small outlined
diamond) erased and replaced by the sparkle emoji (U+2728) as the shield
emblem, to mark the "Neo" fork. A macOS `.icns` is intentionally absent: it is
generated by electron-builder from `icon.png` during a mac build, which is
deferred along with cross-OS packaging and code signing.

### Regenerating

The emblem was produced with Pillow. First the original diamond device is
erased with a white ellipse over the shield interior (the interior is opaque
white; the ellipse stays clear of the black rim). Then the sparkle is rendered
from the system colour-emoji font (Windows `seguiemj.ttf`) and its bounding box
is centred where the diamond was (measured at (420, 247) in the 512 px art). To
reproduce:

```python
from PIL import Image, ImageDraw, ImageFont

base = Image.open("../../../reference/lib/icons/att-512.png").convert("RGBA")
cx, cy = 420, 247  # shield emblem centre in the 512 px art

# Erase the original diamond device (white fill inside the shield rim).
ImageDraw.Draw(base).ellipse((cx - 38, cy - 42, cx + 38, cy + 42),
                             fill=(255, 255, 255, 255))

font = ImageFont.truetype(r"C:\Windows\Fonts\seguiemj.ttf", 109)
scratch = Image.new("RGBA", (218, 218), (0, 0, 0, 0))
ImageDraw.Draw(scratch).text((54, 54), "✨", font=font, embedded_color=True)
glyph = scratch.crop(scratch.getbbox())
target_w = 116  # sits within the shield interior
gw, gh = glyph.size
glyph = glyph.resize((target_w, int(gh * target_w / gw)), Image.LANCZOS)

out = base.copy()
out.alpha_composite(glyph, (cx - glyph.size[0] // 2, cy - glyph.size[1] // 2))
out.save("icon.png")
out.save("icon.ico", sizes=[(16, 16), (24, 24), (32, 32), (48, 48),
                            (64, 64), (128, 128), (256, 256)])
```
