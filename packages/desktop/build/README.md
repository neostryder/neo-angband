# Desktop build resources

electron-builder reads this directory (`directories.buildResources`, the
default `build/`) for application icons.

## Icons

- `icon.png` - 512x512 master, used for Linux and as the source
  electron-builder converts to macOS `.icns` at build time.
- `icon.ico` - multi-size Windows icon (16/24/32/48/64/128/256).

Both are the standard Angband icon (`reference/lib/icons/att-512.png`, the
upstream 4.2.6 art) with the sparkle emoji (U+2728) overlaid at the top-right
to mark the "Neo" fork. A macOS `.icns` is intentionally absent: it is
generated by electron-builder from `icon.png` during a mac build, which is
deferred along with cross-OS packaging and code signing.

### Regenerating

The overlay was produced with Pillow, rendering the sparkle from the system
colour-emoji font (Windows `seguiemj.ttf`). To reproduce:

```python
from PIL import Image, ImageDraw, ImageFont

base = Image.open("../../../reference/lib/icons/att-512.png").convert("RGBA")
W, H = base.size
font = ImageFont.truetype(r"C:\Windows\Fonts\seguiemj.ttf", 109)

scratch = Image.new("RGBA", (218, 218), (0, 0, 0, 0))
ImageDraw.Draw(scratch).text((54, 54), "✨", font=font, embedded_color=True)
glyph = scratch.crop(scratch.getbbox())
target = int(W * 0.42)
gw, gh = glyph.size
glyph = glyph.resize((target, int(gh * target / gw)), Image.LANCZOS)

out = base.copy()
margin = int(W * 0.03)
out.alpha_composite(glyph, (W - glyph.size[0] - margin, margin))
out.save("icon.png")
out.save("icon.ico", sizes=[(16, 16), (24, 24), (32, 32), (48, 48),
                            (64, 64), (128, 128), (256, 256)])
```
